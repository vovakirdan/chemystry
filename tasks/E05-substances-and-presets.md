# Epic 05: Substance Library and Presets

Обязательное условие для всех задач: строгое соблюдение [Project Rules](./00-project-rules.md).

## E05-T01 Реализовать каталог веществ с поиском и фильтрами

**Статус:** done (2026-03-04).

**Контекст:** Каталог веществ нужен для быстрого наполнения реакций.

**Зависимости:** E04-T04, E03-T02.

**Критерии приемки:**
1. Работает полнотекстовый поиск по названию/формуле.
2. Доступны фильтры по фазе и источнику (`builtin/imported/user`).
3. При выборе вещества отображается краткая карточка со свойствами.

**Условия:**
- Поиск и фильтрация должны работать локально без сетевых запросов.

## E05-T02 Реализовать CRUD пользовательских веществ

**Статус:** done (2026-03-04).

**Контекст:** Пользователь должен добавлять и редактировать собственные вещества.

**Зависимости:** E05-T01.

**Критерии приемки:**
1. Доступны операции create/update/delete для user-defined веществ.
2. При удалении вещества есть проверка на использование в сценариях.
3. Ошибки валидации полей показываются до сохранения.

**Условия:**
- Встроенные (`builtin`) вещества не редактируются напрямую.

## E05-T03 Реализовать библиотеку пресетов и загрузку в Builder

**Контекст:** Пресеты — быстрый старт, но редактируются как обычный сценарий.

**Зависимости:** E05-T01, E03-T02.

**Критерии приемки:**
1. Пресеты отображаются списком с метаданными (класс реакции, сложность, описание).
2. Выбор пресета переносит его в ручной конструктор для редактирования.
3. Есть clear feedback, что пользователь работает с копией пресета.

**Условия:**
- Оригинал пресета не должен повреждаться при редактировании сценария.

## E05-T04 Заполнить пресет-пакет по 5 классам реакций

**Контекст:** По PRD в MVP нужен минимум один пресет на каждый класс.

**Зависимости:** E05-T03.

**Критерии приемки:**
1. Добавлены пресеты: неорганическая, кислотно-основная, ОВР, органическая базовая, равновесная.
2. Каждый пресет проходит базовую проверку запуска симуляции.
3. Для каждого пресета есть metadata-описание и образовательная заметка.

**Условия:**
- Формат данных пресетов должен быть версионируемым.
