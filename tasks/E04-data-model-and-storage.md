# Epic 04: Data Model and Local Storage

## E04-T01 Настроить SQLite слой и инфраструктуру миграций

**Статус:** done (2026-03-04).

**Контекст:** Локальная БД — основа offline-first сценария.

**Зависимости:** E02-T01.

**Критерии приемки:**
1. Подключен SQLite драйвер и создан migration runner.
2. Миграции выполняются автоматически при старте приложения.
3. Есть rollback/repair стратегия для неуспешной миграции.

**Условия:**
- Все изменения схемы только через версионированные миграции.

## E04-T04 Реализовать repository API, seed и backup/restore

**Контекст:** Исполнителям нужна прикладная API-обертка поверх SQL.

**Зависимости:** E04-T03.
