# Epic 03: UI Shell and Workspace

Обязательное условие для всех задач: строгое соблюдение [Project Rules](./00-project-rules.md).

## E03-T01 Реализовать каркас 3-панельного интерфейса

**Статус:** done (2026-03-04).

**Контекст:** 3-panel layout — базовый UX-контур из PRD.

**Зависимости:** E02-T04.

**Критерии приемки:**
1. Экран делится на левую, центральную и правую панели с адаптивным ресайзом.
2. Layout стабильно работает в минимальном и максимальном размерах окна.
3. Доступна клавиатурная навигация между основными панелями.

**Условия:**
- Не внедрять бизнес-логику в layout-компоненты.

## E03-T02 Собрать skeleton левой панели (Library / Builder / Presets)

**Контекст:** Левая панель — основной входной поток ручной сборки.

**Зависимости:** E03-T01.

**Критерии приемки:**
1. Реализованы секции-вкладки для `Library`, `Builder`, `Presets`.
2. Добавлены placeholder состояния loading/empty/error.
3. Структура готова к подключению данных без refactor контейнеров.

**Условия:**
- UI должен сохранять состояние активной вкладки между перезапусками.

## E03-T03 Собрать skeleton центральной и правой панелей

**Контекст:** Нужен каркас сцены, таймлайна, контролов условий и расчетов для параллельной разработки.

**Зависимости:** E03-T01.

**Критерии приемки:**
1. В центре есть area для 3D-рендера и control-bar (play/pause/reset/timeline).
2. Справа есть секции `Environment`, `Calculations`, `Summary`.
3. Переходы между секциями не ломают layout и не сбрасывают UI state.

**Условия:**
- Контролы должны быть доступны для automation-тестов (stable selectors).

## E03-T04 Добавить status bar, уведомления и тему

**Контекст:** Для MVP нужны базовые сигналы состояния и единый визуальный стиль.

**Зависимости:** E03-T02, E03-T03.

**Критерии приемки:**
1. Status bar показывает состояние симуляции, профиля точности и FPS-лимита.
2. Система уведомлений поддерживает уровни `info/warn/error`.
3. Внедрены базовые design tokens (цвета, отступы, типографика) и применены глобально.

**Условия:**
- Цветовая схема должна учитывать минимальные требования colorblind-friendly UI.
